# WEBSITE SỰ KIỆN KHOA CÔNG NGHỆ THÔNG TIN

## 1. Giới thiệu đề tài

Đây là website giới thiệu và đăng ký tham gia các sự kiện của Khoa Công nghệ Thông tin.

Website được xây dựng bằng HTML, CSS, JavaScript và Bootstrap nhằm đáp ứng các yêu cầu của bài thi giữa kỳ môn Lập trình Web Frontend.

---

## 2. Công nghệ sử dụng

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* LocalStorage

## 4. Chức năng từng trang

### Trang chủ (index.html)

* Giới thiệu website.
* Hiển thị banner.
* Hiển thị sự kiện nổi bật.
* Điều hướng đến các trang khác.
* Footer liên hệ.

### Trang sự kiện (events.html)

* Hiển thị 8 sự kiện từ JavaScript.
* Tìm kiếm theo tên sự kiện.
* Lọc theo danh mục.
* Lọc theo cấp độ.
* Hiển thị bằng Bootstrap Card.
* Modal xem chi tiết.

### Trang đăng ký (register.html)

* Form đăng ký tham gia sự kiện.
* Kiểm tra dữ liệu bằng JavaScript.
* Thông báo lỗi khi nhập sai.
* Lưu dữ liệu vào LocalStorage.

### Trang danh sách đăng ký (registrations.html)

* Hiển thị danh sách sinh viên đã đăng ký.
* Đọc dữ liệu từ LocalStorage.
* Xóa từng bản ghi.
* Xóa toàn bộ dữ liệu.
