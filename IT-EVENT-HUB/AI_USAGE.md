# AI USAGE REPORT

## 1. Công cụ AI sử dụng

ChatGPT (OpenAI)

---

## 2. Mục đích sử dụng AI

AI được sử dụng để:

* Hỗ trợ xây dựng giao diện website.
* Hỗ trợ viết mã HTML.
* Hỗ trợ viết mã CSS.
* Hỗ trợ viết mã JavaScript.
* Hỗ trợ xây dựng Bootstrap Layout.
* Hỗ trợ tạo dữ liệu mẫu cho các sự kiện.
* Hỗ trợ viết tài liệu README.

---

## 3. Các phần được AI hỗ trợ

### HTML

* Thiết kế cấu trúc trang chủ.
* Thiết kế trang danh sách sự kiện.
* Thiết kế form đăng ký.
* Thiết kế trang quản lý đăng ký.

### CSS

* Thiết kế giao diện.
* Responsive.
* Hiệu ứng Card.
* Banner.
* Footer.

### JavaScript

* Tìm kiếm sự kiện.
* Lọc dữ liệu.
* Hiển thị Modal.
* Validation Form.
* LocalStorage.
* Quản lý danh sách đăng ký.

---

## 4. Các phần tự hoàn thiện

* Chỉnh sửa hình ảnh.
* Thay đổi nội dung phù hợp với đề tài.
* Kiểm tra giao diện.
* Kiểm tra hoạt động của website.
* Đưa website lên GitHub Pages.

AI được sử dụng như công cụ hỗ trợ học tập và phát triển website. Toàn bộ mã nguồn đã được kiểm tra, chỉnh sửa và hoàn thiện trước khi nộp bài.
